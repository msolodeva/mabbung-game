# Tetris 게임

클래식 테트리스 게임의 웹 버전입니다. 모던한 UI와 부드러운 게임플레이를 제공합니다.

## 기능

- 7가지 클래식 테트로미노 블록 (I, O, T, S, Z, J, L)
- 블록 회전 시스템
- 줄 제거 및 점수 시스템
- 다음 블록 미리보기
- 일시정지 기능
- 게임 오버 화면
- 회전, 이동, 줄 제거, 레벨업 등 다양한 효과음
- 90초 제한으로 진행되는 연속 러닝
- 반응형 디자인

## 실행 방법

### 방법 1: 직접 파일 열기
1. `index.html` 파일을 웹 브라우저에서 바로 열기

### 방법 2: 로컬 서버 사용 (권장)
1. 터미널에서 프로젝트 루트 디렉토리로 이동
2. Python 3가 설치되어 있다면:
   ```bash
   python3 -m http.server 8000
   ```
3. 브라우저에서 `http://localhost:8000/tetris/index.html` 접속

## 조작 방법

| 키 | 동작 |
|---|---|
| ← / → | 좌우 이동 |
| ↓ | 빠른 낙하 |
| ↑ | 회전 |
| Space | 하드 드롭 (한 번에 내려오기) |
| P | 일시정지 |

## 점수 시스템

| 줄 제거 | 점수 |
|---|---|
| 1줄 | 100점 |
| 2줄 | 300점 |
| 3줄 | 500점 |
| 4줄 (테트리스) | 800점 |

## 게임 규칙

1. 위에서 떨어지는 테트로미노 블록을 조작하여 바닥에 쌓습니다.
2. 가로 한 줄이 블록으로 가득 차면 그 줄이 제거되고 점수를 획득합니다.
3. 블록이 화면 상단까지 쌓이면 게임 오버됩니다.
4. 더 많은 줄을 동시에 제거할수록 높은 점수를 얻습니다.
